# 🧘 Wellness Analytics Dashboard

## 📌 Overview
A web-based wellness analytics application that helps users track mood, stress, habits, and daily tasks while providing real-time insights for better mental health and productivity.

---

## ✨ Features
- 👤 User Profile Management  
- 😊 Mood Tracking System  
- 📊 Stress Level Monitoring  
- 📝 Task Management (Add/Delete tasks)  
- 🎯 Habit Tracker  
- 📈 Wellness Score Calculation  
- ⏳ Guided Breathing Timer  
- 💬 Motivational Quotes Generator  
- 📊 Mood Analytics Chart  
- 📓 Journal Storage using LocalStorage  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- Chart.js  
- LocalStorage API  

---

## 🚀 How to Run
1. Download or clone the repository  
2. Open `index.html` in any browser  
3. Or use VS Code Live Server extension
